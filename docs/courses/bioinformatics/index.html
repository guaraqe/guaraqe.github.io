                                              Introduction to Bioinformatics

                                              This course has two goals. The first one is to introduce students to bioinformatics, the data it uses and its methods. The second one is to teach students basic programming in R, and to use bioinformatics libraries. At the end of this course, the student should be able to find gene sequences in the internet, and to do simple analysis with them.

                                              Besides the usual grading at each lesson, there will be a periodic practical control after each block of four lectures. It will consist of a test done in the computer where the student will have to execute a task with a visible result using R programming and data sets.

                                              1 Sources

                                              The content of this course is based on A Little Book of R for Bioinformatics by Avril Coghlan and A Little Book of R for Bioinformatics 2.0, by Avril Coghlan, with contributions by Nathan L. Brouwer. They have a Creative Commons Attribution 3.0 License.
