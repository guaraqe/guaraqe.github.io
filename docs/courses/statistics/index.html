                          Statistics using Excel, a data type based approach

                          A practical statistics course for medical students using Excel as the primary analytical tool. The course teaches data analysis through a data-type framework, progressing from categorical to ordered to numerical data. Each type builds upon the previous, adding more sophisticated analytical methods as data properties expand.

                          The course uses medical datasets throughout, teaching histogram creation, probability distributions, correlation analysis, and hypothesis testing. Students learn to avoid common statistical errors by understanding when each analysis method is appropriate for their data type.

                          Designed as both a learning course and reference guide, enabling students to return to specific sections for future analysis work.
